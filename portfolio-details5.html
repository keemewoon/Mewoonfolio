학사관리(MySQL) 데이터베이스 구축 프로젝트

  • 카테고리: Data Base(MySQL)
  • 진행 날짜: 2020. 06. 30

👩‍🔬학사관리(MySQL) 데이터베이스 구축 프로젝트

  • 개발 기간 : 2020. 06. 30
  • 개발 환경 : MySQL, JSP, Tomcat, Eclips, Java
  • 사용 기술 : SQL, Trigger, 함수, View, Index, SubQuery, Join
  • 개발 인원 : 1명
  • 주제 : MySQL기반의 학사관리 정보 시스템 데이터베이스 구축
  • 개요 :
    [ 복잡한 사내 인사업무를 하나로 통합 관리하는 시스템 ]
    본 프로그램을 이용하여 (업무일정 관리, 인사 이동, 월급 관리, 휴가신청, 부서원 근태현황, 공지사항 확인 등) 회사 운영에 필요한 업무를 통합하여 관리할 수 있다.

    [ 개인 맞춤 정보를 제공하는 시스템 ]
    본 프로그램은 부서, 직급 등 사용자 정보에 맞춰 필요한 정보만 선택 제공하여 사용자의 업무의 효율을 증진시킬 수 있다.
  • ERD(논리 데이터베이스)
  • ERD(물리 데이터베이스)
  • 요구사항 명세서
  • 기능소개(JSP와 연동)
    로그인
    ;Login

    - 학번과 비밀번호를 입력 받아 로그인을 할 수 있도록 구현하였으며,
    상단 네비게이션에는 내 정보, 수강정보, 수강신청 페이지로 들어갈 수 있게 배치하였다.
    - 비밀번호가 틀렸을 경우 alert창이 뜨도록 설정하였으며, 로그인이 되지않는다
    - 아이디를 잘못 입력한 경우 존재하지않는 아이디라는 alert창이 뜨며 로그인이 되지않는다.

    내 정보
    ;MyPage

    로그인을 하면 ‘내 정보’페이지가 나온다. 회원가입시 입력 받은 개인정보를 볼 수 있으며 정보 수정 시 키보드와 마우스로 입력 받아 수정 가능하게 구현하였다. 학년, 동아리, 전공은 select box로 변경할 수 있게 구현하였다.

    내 정보 수정
    ;MyPage Edit

    정보수정을 하였더니 다음과 같이 DB에 {김예돌, 1학년 충북충주시, 네버마인드, 골프산업전공} 으로 변경된 것을 확인할 수 있다.

    회원가입
    ;Sign up

    회원가입화면이다. 차례대로 입력 받아 회원가입을 할 수 있게 구현하였으며 수정과 마찬가지로 학년, 동아리, 전공은 select box로 입력 받을 수 있도록 구현하였다. 정보입력 후 회원가입을 누르니 아래와 같이 test 학생의 계정이 생성된 것을 확인할 수 있다.

    탈퇴
    ; Secession

    탈퇴화면이다. 아까 만들었던 test 계정의 학번과 비밀번호를 입력하고 탈퇴버튼을 누르면 아래 DB내역과 같이 test 계정의 회원이 삭제된 것을 확인 할 수 있다.

    로그아웃
    ; Logout

    로그아웃 버튼을 누르면 로그이웃 처리가 되어 로그인 화면으로 돌아가도록 하였다.

    수강신청
    ; Enrolment

    - 수강신청 페이지이다. 내 교과목과 전공이름을 inner join하여 나의 전공과목들만 볼 수 있다. 신청버튼을 누르면 DB에 수강정보가 저장된다.

    - 수강신청이 성공적으로 완료되면 Alert창으로 완료된 것을 알 수 있으며, 수강정보 페이지로 넘어가게 된다.

    - 중복 수강을 막기위해 alert창으로 중복신청 되었다고 알려준다.

    내 수강정보
    ; my class information

    - 내 수강정보 페이지이다. 사용자가 수강신청한 수강내역을 볼 수 있으며 삭제가 가능하도록 구현하였다.

    - 삭제가 성공적으로 완료되면 Alert창으로 삭제가 된 것을 알려준다.
