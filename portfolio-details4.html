사내 관리 시스템

  • 카테고리: Java Console
  • 진행 날짜: 2021. 05. 11

👩‍💻효율적인 업무환경 개선을 위한 사내 관리 시스템

  • 개발 기간 : 2021. 05. 03 ~ 2021. 05. 11 (9일간)
  • 개발 환경 : Eclipse / JDK 1.8 / Googie Drivee
  • 사용 기술 : SQL, Trigger, 함수, View, Index, SubQuery, Join
  • 개발 인원 : 5명
  • 주제 : 파일 입출력 기반의 데이터 처리 자바 콘솔 프로젝트 (회사 인사관리 시스템 자동화 구현)
  • 개요 :
    [ 복잡한 사내 인사업무를 하나로 통합 관리하는 시스템 ]
    본 프로그램을 이용하여 (업무일정 관리, 인사 이동, 월급 관리, 휴가신청, 부서원 근태현황, 공지사항 확인 등) 회사 운영에 필요한 업무를 통합하여 관리할 수 있다.

    [ 개인 맞춤 정보를 제공하는 시스템 ]
    본 프로그램은 부서, 직급 등 사용자 정보에 맞춰 필요한 정보만 선택 제공하여 사용자의 업무의 효율을 증진시킬 수 있다.
  • 기능:
    1. 관리자 기능
      - 직원 관리 : 직원 목록 확인 및 수정, 퇴사관리, 직원등록
      - 휴가 관리 : 직원 휴가신청 리스트 확인, 휴가 승인, 반려
      - 급여 관리 : 직급별 급여 조회, 수정
      - 공지 사항 : 공지사항(부서별, 직급별) 등록, 수정, 삭제
      - 건의 사항 : 건의사항 확인, 수정, 삭제
    2. 사용자 기능
      - 마이페이지 : 개인정보, 급여현황, 퇴사신처을 관리 하는 기능
      - 부서원 정보 : 사용자가 속한 부서의 부서원 정보를 전부 보여주는 기능
      - 근태 관리 : 출근/퇴근/외출/외근 처리를 하는 기능
      - 회사 일정 : 부서별 회사일정을 확인 하고 부서별 일정을 추가해주는 기능
      - 휴가 관리 : 사용자의 휴가일 수, 휴가 신청, 휴가 현황을 관리하는 기능
      - 건의 사항 : 직접 건의사항을 작성할 수 있는 기능
      - 공지 사항 : 부서별/전체 공지사항을 선택하여 확인 할 수 있는 기능
  • 담당기능: 관리자 - 직급별/부서별 직원 조회, 직급/부서 수정, 퇴사 직원 관리 직원 추가 기능 구현
  • 기능소개
    공지사항
    ;Notice

    세부내용 : 사용자가 속한 부서와 전체 공지사항을 확인 가능하다.
    공지사항의 갯수가 많다면 페이지 이동으로 내용을 확인 할 수 있게 구현하였다.
    사용기술 : ArrayList, BufferedReader, Calendar

    급여내역 확인
    ;Salary details

    세부 내용 : 사용자의 급여 내역을 자세히 확인 할 수 있으며, 세금 공제 내역과, 현재 연봉, 추가 야근 수당을 확인 할 수 있다.
    사용 기술 : ArrayLIst, BufferedReader

    부서별 캘린더
    ;Calendar

    세부 내용 : 페이지 이동을 하여 월별로 확인 가능하며, 원하는 날짜에 부서별 회사일정을 추가 할 수 있다.
    사용기술 : ArrayList, BufferedReader, Calendar

  • 🙋‍♀️프로젝트를 진행하며 배운점
    콘솔 프로그램이라는 한계가 있기 때문에
    팀원들과 어떻게하면 사용자의 요구사항을 콘솔프로그램이라는 특성에 맞게
    화면에 출력하게끔 할 수 있을까에대한 회의를 많이 했던 프로젝트였다.
    ArrayList와 BufferedReader 등 자바의 기능에 대한 지식을 프로젝트를 확인 할 수 있었다.
