  • 개발 기간 : 2021. 06. 01 ~ 2021. 06. 05 (5일간)
  • 개발 환경 : Oracle
  • 사용 기술 : SQL, Trigger, 함수, View, Index, SubQuery, Join
  • 개발 인원 : 5명
  • 주제 : 교육센터 운영 데이터베이스 도와주는 웹사이트
  • 개요 : 쌍용 교육센터를 운영하면서 필요한 모든 사용자의 데이터베이스를 구축하시오.
  • 요구사항:
  • 기능:

    2. 교사 기능
      - 강의스케줄 조회, 배점 입출력, 성적 입출력 기능
